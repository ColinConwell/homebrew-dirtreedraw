# Homebrew Tap for DirtreeDraw

This is the official Homebrew tap for [DirtreeDraw](https://github.com/colinconwell/DirTreeDraw).

## Installation

```bash
brew install colinconwell/dirtreedraw/dirtreedraw
```

Or tap first, then install:

```bash
brew tap colinconwell/dirtreedraw
brew install dirtreedraw
```

## Usage

After installation, run:

```bash
dirtreedraw
```

See the [main repository](https://github.com/colinconwell/DirTreeDraw) for full documentation.

## Updating

```bash
brew upgrade dirtreedraw
```

## Uninstalling

```bash
brew uninstall dirtreedraw
brew untap colinconwell/dirtreedraw
```
